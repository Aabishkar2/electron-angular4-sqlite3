# Community Manager

## Setup
```
git clone git@github.com:leota/electron-angular4-sqlite3.git
cd electron-angular4-sqlite3
npm install
```

#### Copy database file
On Linux/Mac:

```
npm run db
```
On Windows:
```
npm run db-win
```
## Build
```
npm run rebuild
```

## Run

On Linux/Mac:

```
npm run electron
```

On Windows:
```
npm run electron-win
```
